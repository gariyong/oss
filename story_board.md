# Axiom Rift

## 세계관 개요 (Worldbuilding Overview)
■ 아크시온(Axion) 우주

현재 우주는 보이지 않는 규칙들. 시간, 공간, 인과, 질서들이  하나의 핵심체(Axiom Core)에 의해 유지되고 있다.
Axiom은 물리 법칙이 아니라, '우주가 스스로를 정돈하기 위해 만든 규범적 시스템'이며,
관측자만이 이 규칙을 인지하고 다룰 수 있다.

● Axiom의 구성 요소

시간의 규율 — 흐름, 역순, 정지

공간의 규율 — 거리, 위치, 경계

인과의 규율 — 원인 → 결과

존재의 규율 — 파괴/전이/보존

선형성 규율 — “턴제 전투”가 성립하는 이유

Axiom Core가 사라지면 이 규칙들이 분열해 균열(Rift) 로 흘러나와 괴현상을 만든다.

----
## 관측소(Observatorium)
■ 목적

관측소는 우주의 규칙을 감지하고 기록하며, 이상 현상을 봉합하기 위한 기관이다.
외형은 연구시설이지만, 기능적으로는 규칙을 해석하는 거대한 감각기관에 가깝다.

----

## 관측자(Observer)의 역할

관측자는 보통의 생명체와 다르게 규율의 흐름을 시각·촉각처럼 직접 인식할 수 있다.

● 관측자의 능력

규칙 읽기 : 공간·시간 왜곡을 감지

규칙 수정 : 작은 균열을 봉합

규칙 위반(궁극기) : 일시적으로 규율 “밖”에서 행동

→ 이것 때문에 재작자는 플레이어를 동류라고 여긴다.

● 플레이어의 성격

겁은 나지만, 비꼬듯 농담하며 긴장을 푸는 타입

규칙을 부수는 것을 두려워하면서도 "필요하다면 해낼 마음"은 있음

재작자에게 말대꾸 잘함

----

## 주요 등장 캐릭터 설정
■ 리네아 (Linea)

관측소 보좌관

차분하고 실용적이며, 감정보다는 상황 판단을 우선

플레이어를 믿는 이유:
"규칙을 읽는 감각이 나와는 다른 종류야. 네가 필요해."

■ 재작자(Rewriter)

과거 “규칙을 다시 쓰는 실험 프로젝트”의 잔재

Axiom이 만들어낸 규율에 갇힌 존재들(시간정지 고아, 공간루프 생명체 등)을 보며 규칙 자체에 회의를 품음

"자유는 규칙 밖에서만 존재한다"가 그의 철학

● 동기

"우주가 유지되는 동안, 누군가는 항상 '틀 밖의 희생자'가 있었다."

그래서 아크시옴 코어를 파괴하며 기존 규율을 해방시키려 함

악이 아니라 철학적 반항에 가까운 존재.

■ 플레이어

신규 관측자

규율에 특별한 감응 능력을 가진 예외적 존재

규칙을 깨는 힘(궁극기)을 갖고 있지만,
그 힘이 위험하다는 걸 본능적으로 알고 있음

## 챕터 1 — 침묵한 규율

 ■ 오프닝

(관측소 내부, 붉은 경보음. 통로가 짧게 ‘끊겼다’ 다시 이어진다.)

리네아 : "신입 관측자. 환영 인사는… 일단 보류할게. 지금 상황이 좋지 않아."

플레이어 : "경보음이 계속 울리는데… 이거 환영 이벤트는 아니죠?"

리네아 : "규율 코어(Axiom Core)가 ‘응답 없음’ 상태야.
우주의 중심 규칙이 사라졌다는 뜻이지."

플레이어 : "저… 처음 출근했는데 우주가 고장난 건가요?"

리네아 : "간단히 말하면 그래. 네 첫 임무는… 생존."

---
■ 탐색 중

(복도, 시간이 0.2초 멈춘 뒤 튐)

플레이어 : "방금… 시간 3초 정도 도망갔는데요?"

리네아 : "시간 규율이 완전히 꼬였어."

플레이어 : "저 아직 초과근무도 안 했는데요?"

리네아 : "농담할 여유 있네. 좋아. 그 감각 유지해.
넌 관측자야. 규칙을 읽고… 필요하다면 ‘부숴서’ 해결해."

■ 전투 직전

리네아 : "저기! 시간 균열 개체 접근! 전투 시스템 가동!"

플레이어 : "하필 첫날에요!? 뭐, 깨진 시간 조각한테 지긴 싫지만!"

---

## 챕터 2 — 균열의 첫 조각

■ 균열 발견

(공간에 푸른 틈이 열리고, 주변 그림자가 시간차처럼 어긋남)

플레이어 : "와… 이게 균열(Rift)인가요? 생각보다… 살아 움직이는 건데요?"

리네아 : "아크시옴이 찢어질 때 생긴 상처야."

플레이어 : "저 안 들어가야 하죠? 들어가면 뒤틀릴 것 같은데요?"

리네아 : "여기 제정신인 사람은 너뿐이야."

플레이어 : "그게 더 문제 같다고요…"

----

■ 균열 내부

(한 걸음 → 두 걸음 전진하는 듯한 어긋남)

플레이어 : "으악! 1m 걸었는데 3m 나갔어요! 공간이 미끄러져요!"

리네아 : "심박수 올라갔어. 괜찮아?"

플레이어 : "괜찮지 않아요! 공간이 기분 나쁘게 숨 쉬고 있어요!"

리네아 : "균열 안에선 규칙이 없어. 하지만 넌 관측자의 감각을 믿어."

----
■ 전투 전

플레이어 : "저건… 아예 순서를 안 지키는데요?"

리네아 : "네 턴을 기다리지 않는 놈이야."

플레이어 : "그건 반칙 아닌가요?"

리네아 : "반칙을 쓰려면 반칙을 알아야지."

----

■ 아크시옴 잔광 발견

(빛의 파편이 떠다님)

플레이어 : "이건… 아크시옴의 조각?"

리네아 : "조각났다는 건 누군가 의도적으로 뜯어갔단 뜻이야."

플레이어 : "우주의 규칙을 누가 훔쳐가요?"

----
## 챕터 3 — 규칙을 거부하는 자
■ 재작자(Rewriter) 등장

재작자(목소리) : "관측자… 따라왔군."

플레이어 : "당신이 아크시옴을… 훔친 사람?"

재작자 : "훔친 게 아니다.
나는… 해방시켰다."

플레이어 : "우주가 무너지고 있어요!"

재작자 : "무너지는 건 ‘누군가를 가둔 낡은 규칙’일 뿐."

----
■ 분석

리네아 : "그는 재작자(Rewriter).
규칙을 다시 쓰려는 자."

플레이어 : "왜 규칙을 미워하는 거죠?"

리네아 : "그는… 규율에 희생된 존재들을 봐왔어.
시간에 갇힌 아이들, 공간이 갈라져 사라진 마을…"

----
■ 중간 보스 직전

재작자 : "너의 궁극기… 규칙 밖의 힘이지."

플레이어 : "그렇다고 규칙을 부수고 싶진 않아요!"

재작자 : "규칙은 깨라고 있는 것이다."

플레이어 : "저는… 지킬 가치가 있는 규칙도 있다고 생각해요!"

----
■ 중간 보스 후

플레이어 : "하아… 힘드네요…"

리네아 : "그의 신호가 최심부 균열로 향해."

플레이어 : "거기서 끝을 낼 수 있겠죠…?"

----
## 챕터 4 — Axiom Rift

(바닥이 기울어지고 인과가 파도처럼 흔들림)

플레이어 : "아크시옴이… 이렇게 찢어져…"

재작자 : "보이는가? 이것이야말로 자유다."

플레이어 : "전… 이걸 붕괴라고 부르는데요?"

----
■ 최종 전투 직전

재작자 : "너도 규칙 밖에서 힘을 쓰고 있으면서."

플레이어 : "전… 규칙을 ‘지키기 위해’ 깨는 거예요!"

재작자 : "규칙은—깨라고 있는 것이다!"

플레이어 : "그 말… 이제 제 방식으로 돌려드릴게요."

----
■ 최종 전투 후

재작자 : "흥미롭군…
네가 선택한 규칙이… 다음 우주를 이끌지도 모르지…"

----
■ 1장 엔딩

리네아 : "아크시옴이 안정됐어. 우주도 회복될 거야."

플레이어 : "저… 규칙을 지킨 건가요? 아니면 부쉈나요?"

리네아 :
"그건 네가 판단할 문제.
우주는… 규칙대로만 움직이지 않아."

플레이어 : "…다음 균열이 나타나면 제가 막을게요."

리네아 :
"기대할게, 관측자."

리네아 : "기대할게, 관측자."
